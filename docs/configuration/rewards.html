Rewards

You can set multiple rewards with a message, command, items and money (require Vault).

First join rewardsâ€‹

Reward your new players.
Available placeholders are: %player% (the new player)

first-join-rewards:
  # Enables the feature
  enabled: false
  type: single
  # It can be any word or number
  1:
    # Require vault
    money: 35
    # This is an example of random commands as explained in https://uw.roch-blondiaux.com/docs/configuration/important
    command:
      - "/say hello %player%"
      - "/say hi %player%"
    # Message sent only to the player
    message: "First join yea!"
    # A list of items given to the player
    items:
      # It can be any word or number
      sword:
        name: "&aMagic Sword"
        material: STONE_SWORD
        amount: 3
        lore:
          - "&aHey !"
          - "&bI'm a magic"
          - "&cSword"
        enchants:
          sharpness: 1

Types:
single - Only the first reward will be given.
random - A random reward will be selected and given from the configured ones.
all - All rewards will be given.

Milestonesâ€‹

With the welcome command, players earn a point each time they welcome someone.
You can set milestones and reward them.

The features are equal to the first join reward, except that you have to specify the number of points for the milestone.

Available placeholders: %player% (the player who achived the milestone)

# "points" is required
# One between "command", "money" and "items" is required
rewards:
  1:
    money: 50
    message: "%prefix% &a&lOhhhhh ! Good job you earn $50 by welcoming 5 players"
    points: 5
  2:
    command: "/give %player% diamond 1"
    message: "%prefix% &a&lNice job ! You earn a diamond by welcoming 10 players"
    points: 10
  3:
    points: 15
    items:
      sword:
        name: "&aMagic Sword"
        material: STONE_SWORD
        amount: 3
        lore:
          - "&aHey !"
          - "&bI'm a magic"
          - "&cSword"
        enchants:
          sharpness: 1
