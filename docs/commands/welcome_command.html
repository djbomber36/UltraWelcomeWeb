Welcome command

Allows players to welcome new ones and receive a point every time.

You can configure it in the config:

# A restart of the server is needed if you change one of this parameter
welcomeCommand:
    # The name of the command (the feature can be disabled by putting this blank)
    command: "welcome"
    # Permission for the command (disabled if blank)
    permission: ""
    # Command aliases (must be a list)
    aliases:
        - "w"
    # The player needs to welcome the new one in this time range (in seconds)
    time-limit: 12

Subcommands​

Top

/welcome top displays a leaderboard of the players points.

The message style is configurable in the config:

top:
  header: "%prefix% &7Top welcomers : "
  # Available placeholders: %rank% & %rankedPlayer% & %points%
  format: "&7N°&a%rank% - &a%rankedPlayer%&7 &b%points%&7 points"
  # Defines how many players should be displayed in the leaderboard
  length: 10
Help

/welcome help displays the help message for the welcome command.

The message is configurable in the language file.
