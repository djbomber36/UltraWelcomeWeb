Languages

You an change the language by changing the lang parameter in the config.
However, you cannot put any value, it is necessary that the value of the 'lang' parameter is equal to the name of your language file in the UltraWelcome/lang folder.

# The name of the language file without the ".lang" extentionlang: en_US

Download your language without having to translate it:

English - Sharkz
France - Sharkz
German - xItsMyTimex
Italian - Quantum
Hungarian - Victor75007

Your language does not appear on this page?
Submit your translation by sending me a message on spigotmc.org or by opening a ticket on the official Discord server
