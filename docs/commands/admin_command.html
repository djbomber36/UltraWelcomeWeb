Admin command

Command: ultrawelcome Aliases: uw

All messages are configurable in the language file.

/ultrawelcome - displays the version of the plugin (no permission needed).
/uw reload - reloads the configuration and language files (permission: ultrawelcome.reload).

/uw points - manage player points (permission: ultrawelcome.points).
Subcommands:
get (player) - get player points.
add (player) (points) [giveReward] - add player points.
set (player) (points) [giveReward] - set player points.
remove (player) (points) - remove player points.
() value is needed
[] value is optional
giveReward is a true/false and defines if the player should receive rewards between his old points number and his new one.

All the messages can be set in the language file.
Available placeholders for the points subcommands: %player% & %points% & %current_points% (not for the get subcommand)
