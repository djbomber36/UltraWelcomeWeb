Actions

Execute actions on welcome, join and first join.
Send a broadcast, spawn a firework, execute a command and more.

Every action can be single (the action is the same for everyone) or per-group (specify multiple actions with permissions)

Welcome actions​

Actions ran when a player use the Welcome command

Placeholders available: %player% (player who executed the command) and %new_player% (player who just joined for the first time)

Broadcast
broadcast:
  enabled: true
  type: global
  sender: broadcast
  # SINGLE
  broadcast: "Broadcast"
  # PER-GROUP  
  broadcasts:
    1:
      message: "Hi"
      permission: test.simplemsg

Types:
global - Send broadcast to all online players
single - Send broadcast to command sender and new player

Senders:
broadcast - Message is broadcasted
chat - Message is sent in chat by the command sender

To add a per-group broadcast you need to specify a number/name and inside there must be a message and a permission

Command
command:
  enabled: true
  type: global
  # SINGLE
  command: "/say Hello %player% and %new_player%"
  # PER-GROUP
  commands:
    1:
      command: ""
      permission: a.a

Types:
global - %player% placeholder is changed to every online player name
single - %player% placeholder is changed to command sender name

%new_player% is always the new player and doesn't change with the type.

Firework
firework:
  enabled: true
  type: global
  # SINGLE
  firework:
    power: 3
    flicker: true
    trail: true
    type: STAR
    colors:
      base:
        - WHITE
        - LIME
      fade:
        - RED
        - BLACK
  # PER-GROUP
  fireworks:
    1:
      firework:
        power: 3
        flicker: true
        trail: true
        type: STAR
        colors:
          base:
            - WHITE
            - LIME
          fade:
            - RED
            - BLACK
      permission: a.a

Types: global - Spawn firework to all online players single - Spawn firework to command sender and new player

Title
title:
  enabled: true
  type: global
  # SINGLE
  title:
    title: "Hello"
    subtitle: "there!"
    fade-in: 10
    stay: 20
    fade-out: 10
  # PER-GROUP
  titles:
    1:
      title: "Hello"
      subtitle: "there!"
      fade-in: 10
      stay: 20
      fade-out: 10
      permission: a.a

Types: global - Shown to all online players single - Shown to command sender and joined player

Fade and Stay are in ticks, so 20 ticks equal 1 second.

Reward
reward:
  enabled: true
  # SINGLE
  reward:
    money: 0
    message: "&a&lGood Job ! &7You earned $0 for welcoming &a&l%new_player%"
  # PER-GROUP
  rewards:
    1:
      money: 0
      message: ""
      permission: a.a

The reward is just money only for the command sender.

First join actions​

Actions ran when a player joins the server for the first time

It’s equal to the Welcome Actions.
The difference is that there isn’t the reward, there isn’t the sender option for the broadcast and the only placeholder available is %new_player% (also %player% if you use the command in global mode).

Join actions​

Equal to the First Join Actions.

Only difference is %new_player% placeholder is %joined_player%.
